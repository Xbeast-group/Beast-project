# 📄 AI Terms & Conditions Summarizer

> **Understand lengthy Terms & Conditions in seconds using AI.**

An AI-powered web application that transforms complex Terms & Conditions into concise, easy-to-understand summaries. Instead of spending time reading legal documents, users receive a structured overview highlighting the most important clauses, risks, and key points.

---

## ✨ Features

* 📑 Paste any Terms & Conditions or Privacy Policy
* 🤖 AI-generated concise summaries
* ⚡ Fast and accurate processing
* 🎯 Highlights important clauses
* 📱 Responsive and user-friendly interface
* 🔒 Privacy-focused (no unnecessary data storage)
* 🌐 Clean modern UI

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Xbeast-group/Beast-project.git
```

### Navigate into the project

```bash
cd Beast-project
```

### Install dependencies

```bash
npm install
```

### Create an environment file

Create a `.env` file in the root directory.

Example:

```env

```

### Run the application

```bash
npm start
```

or

```bash
node server.js
```

The application will start on:

```
http://localhost:3000
```
